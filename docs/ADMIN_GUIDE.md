# Руководство администратора MyShop

## Вход в админку

1. Откройте `https://your-domain.com`
2. Нажмите "Войти" в верхнем меню
3. Введите: `admin` / `admin123`
4. Перейдите в раздел "Админ" (видна только для админов)

## Управление товарами

### Добавление товара

1. Перейдите в "Админ" → "Товары"
2. Нажмите "Добавить товар"
3. Заполните форму:
   - **Название** — отображается в каталоге
   - **Цена** — в рублях
   - **Изображение** — URL картинки или загрузите файл
   - **Категория** — electronics, storage, accessories, wearables, gaming, smart_home
   - **Бренд** — название бренда
   - **Рейтинг** — от 0 до 5
4. Нажмите "Сохранить"

### Редактирование

1. Найдите товар в списке
2. Нажмите "Редактировать"
3. Измените нужные поля
4. Нажмите "Сохранить"

### Удаление

1. Найдите товар
2. Нажмите "Удалить"
3. Подтвердите удаление

## Управление пользователями

### Просмотр списка

1. Перейдите в "Админ" → "Пользователи"
2. Вы увидите всех зарегистрированных пользователей

### Изменение роли

1. Найдите пользователя
2. Выберите новую роль:
   - **user** — обычный пользователь
   - **admin** — администратор
3. Нажмите "Сохранить"

## Управление заказами

### Просмотр заказов

1. Перейдите в "Админ" → "Заказы"
2. Вы увидите все заказы с деталями

### Изменение статуса

1. Найдите заказ
2. Выберите новый статус:
   - **pending** — ожидает обработки
   - **processing** — в обработке
   - **shipped** — отправлен
   - **delivered** — доставлен
   - **cancelled** — отменён
3. Нажмите "Сохранить"

## Управление промокодами

### Создание промокода

1. Перейдите в "Админ" → "Промокоды"
2. Нажмите "Добавить промокод"
3. Заполните форму:
   - **Код** — буквами, например SALE20
   - **Скидка** — процент от 1 до 100
   - **Срок действия** — дата окончания
   - **Макс. использований** — 0 = безлимит
4. Нажмите "Сохранить"

### Деактивация

1. Найдите промокод
2. Переключите "Активен" в выкл
3. Сохраните

## Статистика

На главной странице админки отображается:

- **Пользователей** — общее количество
- **Товаров** — в каталоге
- **Заказов** — всего
- **Выручка** — сумма всех заказов

## Загрузка изображений

### Через админку

1. Перейдите в "Админ" → "Товары"
2. Нажмите "Добавить товар"
3. В поле "Изображение" нажмите "Загрузить файл"
4. Выберите файл (JPG, PNG, WebP, GIF)
5. Максимальный размер: 5 МБ

### Через URL

1. Загрузите картинку на любой хостинг (imgur.com, etc.)
2. Скопируйте прямую ссылку
3. Вставьте в поле "Изображение"

## Часто задаваемые вопросы

### Как сменить пароль администратора?

В настоящее время смена пароля доступна через профиль пользователя. В будущих версиях будет отдельная настройка.

### Как добавить новую категорию?

Категории задаются в коде. Для добавления новой обратитесь к разработчику.

### Как посмотреть статистику продаж?

В версии Pro доступен раздел "Аналитика" с графиками продаж, товарами и конверсией.

### Как интегрировать с маркетплейсами?

Доступно в версии Enterprise. Обратитесь к разработчику для настройки интеграции с Wildberries/Ozon.
